# biTurbo v0.2.0

## Reliability

- Added transactional, versioned SQLite migrations with automatic pre-migration backups and rollback.
- Added durable vector mutation journaling and crash replay so SQLite, FTS, turbovec indexes, counters, and activity stay aligned.
- Added cross-process project locks for safe concurrent GUI and MCP writes.
- Embedding-model changes now rebuild a shadow index under supervision and activate it atomically.

## Supervised operations

- Ingest, multi-project ingest, consolidation, model rebuilds, and watched indexing are persisted operations.
- Operations support status, listing, cancellation, checkpoints, retry-safe restart recovery, and coalesced watch reruns.
- Existing progress events and synchronous MCP behavior remain compatible.

## Recall and MCP

- Tauri and MCP now share transport-neutral application behavior for validation, stats, bootstrap, activity, and agents.
- Added asynchronous ingest and operation-control interfaces.
- Added explainable recall with vector/FTS ranks, matched terms, bounded local feedback boosts, and useful/not-useful controls.
- Existing search, recall, language ingestion, MCP inputs, defaults, and response fields remain compatible.
