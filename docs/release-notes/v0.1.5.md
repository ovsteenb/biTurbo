# biTurbo v0.1.5

## What's New

- **Auto-updater.** biTurbo now checks for new versions automatically. Open Settings → Updates to check for the latest version and install it with one click. Updates are downloaded, verified, and installed without leaving the app.

## Improvements

- Added "Updates" section in Settings with check-for-updates and install buttons
- Release notes for new versions are shown directly in the app before installing
