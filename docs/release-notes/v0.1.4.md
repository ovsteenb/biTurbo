# biTurbo v0.1.4

## What's New

- **MCP server now bundled inside the app.** The `biturbo-mcp` memory server is included in every installer — no separate build step needed after installation.
- **Windows installer available.** Download `biTurbo_0.1.4_x64-setup.exe` and run it to install biTurbo on Windows.
- **macOS installers for both Intel and Apple Silicon.** Download the `.dmg` that matches your Mac:
  - Apple Silicon (M1/M2/M3/M4): `biTurbo_0.1.4_aarch64.dmg`
  - Intel Macs: `biTurbo_0.1.4_x64.dmg`

## Requirements

- **Windows**: Windows 10 or later (WebView2 is pre-installed on Windows 11)
- **macOS**: macOS 11 (Big Sur) or later
